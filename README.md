# MyProject
It's a online food delivery front end design using HTML, CSS and JS.
Overview
A simple frontend web application for ordering food online.
Users can browse menus, add items to cart, and view order details through a clean and responsive interface.

Features
Clean and user-friendly UI
Menu listing
Add to cart functionality
Responsive design

Technologies Used
HTML
CSS
JavaScript

How to Run
Open index.html in any web browser.

Conclusion
This project demonstrates basic frontend development skills with a focus on usability and clean design.
